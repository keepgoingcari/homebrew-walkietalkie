# homebrew-walkietalkie

Homebrew tap for installing `walkietalkie` on macOS.

## Install
```bash
brew tap keepgoingcari/walkietalkie
brew install --cask walkietalkie
```

Then run setup:
```bash
walkietalkie setup
```

## Upgrade
```bash
brew update
brew upgrade --cask walkietalkie
```

## Uninstall
```bash
brew uninstall --cask walkietalkie
brew untap keepgoingcari/walkietalkie
```

## What is Walkietalkie?
Walkietalkie is a push-to-talk voice layer for coding in terminal workflows.
It lets you speak and inject text/prompts into your focused app.

Main project: https://github.com/keepgoingcari/walkietalkie
